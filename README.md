# Oque foi utilizado?
- HTML:
  - Criação do visor de resultado com input text.
  - Criação de input button para leitura de dados pelo javascript
  - Criação de div para centralizar a calculadora
  - Criação de icone para o site
- CSS:
  - Criação e manipulação de elementos da pagina.
  - Upload de fotos para o background 
- JavaScript:
  - Criação de funções:
    - Função do display/Visor
    - Função de cálculo com o metodo eval
    - Função de apagar o ultimo número
    - Função de limpar o visor
  - Tratamento de erros com condicionais:
    - O codigo não permite utilizar dois operadores:
      - Ex: 10++20
      - Exceto pelo operador ** , no caso seria o exponencial
    - O codigo não permite utilizar dois pontos seguidos:
      - Ex: 10..20
